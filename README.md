# Bug Bounty Report: XSS & IDOR Vulnerabilities

## Overview
This repository contains a bug bounty report identifying critical vulnerabilities in a demo banking application.

## Vulnerabilities Found
- Cross-Site Scripting (XSS)
- Insecure Direct Object Reference (IDOR) via HTTP Parameter Tampering

## Summary
The vulnerabilities allow attackers to:
- Access sensitive user data
- Manipulate transactions
- Perform unauthorized actions

## Tools Used
- Burp Suite
- Browser DevTools
